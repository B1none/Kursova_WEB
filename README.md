# Курсова робота: Marvel Rivals Heros

## 📋 Описання проекту

Веб-документ "Marvel Rivals Heros" - це інформаційна система для перегляду, фільтрації та управління **47 героями** гри Marvel Rivals. Проект розроблений з використанням HTML, CSS та JavaScript відповідно до вимог курсової роботи.

**Технології:**
- HTML5 (структурна розмітка)
- CSS3 (оформлення та адаптивність)
- JavaScript (динаміка та інтерактивність)
- LocalStorage (збереження даних)

---

## 🎯 Основні функціональності

### 1. **Каталог героїв**
- Відображення **47 героїв Marvel Rivals** з реальними фотографіями
- Кожен герой має: ім'я, тип, опис, рівень сили, здібності, фотографію
- Красиве оформлення з реальними зображеннями героїв

### 2. **Система пошуку та фільтрації**
- **Пошук за іменем** - реал-тайм пошук героїв англійською та українською
- **Фільтр за типом** - Vanguard, Duelist, Caster, Strategist
- **Фільтр за силою** - розбивка на 4 категорії (1-2, 3-5, 6-8, 9-10)

### 3. **Улюблені герої**
- Додавання/видалення героїв з улюблених
- Збереження улюблених у localStorage
- Окрема сторінка для перегляду улюблених

### 4. **Порівняння героїв**
- Вибір двох героїв для порівняння
- Відображення типу, сили, кількості здібностей
- Список здібностей кожного героя

### 5. **Система голосування**
- Голосування за улюбленого героя
- Збереження результатів у localStorage
- Лічильник голосів для кожного героя

### 6. **Статистика**
- Всього героїв в каталозі
- Кількість улюблених
- Найпопулярніший герой (за силою)

### 7. **Модальне вікно з деталями**
- Детальна інформація про героя
- Список всіх здібностей
- Рівень сили зі шкалою
- **НОВИНКА:** Окрема сторінка `hero-details.html` для повної інформації про персонажа

### 8. **Детальна сторінка персонажа** ✨ НОВИНКА
- Повна історія персонажа
- Великі фотографії героїв (300x300px+)
- Розширена інформація про здібності
- Дата випуску персонажа в грі
- Пов'язані персонажи
- Красива сторінка з детальним дизайном

## 💾 Збереження даних

Проект використовує **localStorage** для збереження:
1. **Улюблені герої** - `favoriteHeroes` key
2. **Результати голосування** - `voteData` key

Дані зберігаються автоматично і відновлюються при перезавантаженні сторінки.

**Очистка даних:** Відкрити консоль (F12) і виконати `clearAllData()`

## 📌 Алгоритми та методи

### Пошук героя
```javascript
// Лінійний пошук за іменем
heroes.filter(hero => hero.name.toLowerCase().includes(searchValue))
```

### Фільтрація
```javascript
// Комбінована фільтрація за типом та силою
filtered = heroes.filter(hero => 
    matchesSearch && matchesType && matchesPower
)
```

### Порівняння героїв
```javascript
// Паралельне відображення характеристик
compare(hero1, hero2) // Виводить карточки поруч
```

### Голосування
```javascript
// Накопичення голосів у об'єкті
voteData[heroId].votes++
```
